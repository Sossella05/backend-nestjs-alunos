# Back-end - Cadastro de Alunos (NestJS)

Este é o back-end do sistema de cadastro de alunos, construído com NestJS, usando uma lista em memória (sem banco de dados).

## 🔧 Tecnologias
- NestJS
- TypeScript
- Node.js

## 📦 Funcionalidades
- Criar aluno
- Listar alunos
- Buscar aluno por ID
- Atualizar aluno
- Remover aluno

## ▶️ Como executar

1. Instale as dependências:
npm install

2. Execute o projeto:
npm run start:dev
